# day-ui

基于 Vue3 的组件库，样式借用 element-plus 的，用于学习使用

## 安装

```
npm install -S day-ui
```

### 引入

```
import DayUI from 'day-ui'
import 'day-ui/lib/style/index.css'

createApp(App).use(DayUI)
```

> 用法可查看 example 目录

[文档地址](https://bluestar123.github.io/day-ui-docs/)
